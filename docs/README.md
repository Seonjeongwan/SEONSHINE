# 문서 가이드

이 디렉토리는 프로젝트 문서를 카테고리별로 정리한 곳입니다.

## 📁 디렉토리 구조

### 📦 deployment/ (배포 관련)

배포 및 운영 환경 관련 문서

- **DEPLOYMENT_GUIDE.md**: 배포 가이드 (체크리스트, 재시작 순서 포함)
- **DEPLOYMENT_STRATEGY.md**: 배포 전략 (브랜치 전략, 환경별 차이점 포함)

### 🛠️ development/ (개발 관련)

개발 환경 및 인프라 개선 관련 문서

- **LOCAL_DEVELOPMENT.md**: 로컬 개발 환경 가이드
- **INFRASTRUCTURE_IMPROVEMENTS.md**: 인프라 및 Docker 개선점 분석

### 🗄️ database/ (데이터베이스 관련)

데이터베이스 관련 문서

- **DB_RUN_GUIDE.md**: 데이터베이스 실행 가이드
- **MYSQL_SYNTAX_GUIDE.md**: MySQL 문법 가이드

### 🧪 testing/ (테스트 관련)

테스트 관련 문서

- **TESTING_CHECKLIST.md**: 테스트 체크리스트
